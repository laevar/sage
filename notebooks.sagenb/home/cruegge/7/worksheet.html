Aufgabe 8 *

Schreiben Sie eine Prozedur, die eine Nullstelle einer gegebenen Funktion $f:[a_0,\,b_0]\to\mathbb{R}$ mit $f(a_0)\cdot f(b_0)<0$ zu einer gegebenen Genauigkeit $\mathrm{TOL}$ berechnet. Betrachten Sie hierzu eine Folge von Intervallen $[a_k,\,b_k]$, die sich aus der folgenden Vorschrift ergeben:

Wiederhole bis $|b_k - a_k| \leq 2*\mathrm{TOL}$:

  • Berechne $g:=f\left(\frac{a_k+b_k}{2}\right)$.
  • Gilt $f(a_k) * g<0$, so definiere $a_{k+1}=a_k,\,b_{k+1}=\frac{a_k+b_k}{2}$.
  • Andernfalls definiere $a_{k+1}=\frac{a_k+b_k}{2},\,b_{k+1}=b_k$.

Testen Sie die Prozedur an $f(x)=x^3-2$, $[a_0, b_0] = [0,3]$ für $\mathrm{TOL}=10^{-4}$ und $\mathrm{TOL}=10^{-6}$. Verwenden Sie maximal 100 Iterationen.

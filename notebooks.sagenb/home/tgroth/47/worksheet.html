Aufgabe 1

{{{id=1| def f(x): if x>1: return sqrt(x) return x^2 /// }}}

Aufgabe 4

Bestimmen sie den Differenzenquotient einer beliebigen analytischen Funktion an $n$ Stellen in einem vorgegebenen Intervall $[a,b] , a<b \in \mathbb{N}$.

{{{id=7| #erwartet eine Funktion in Abhängigkeit von einer Variablen def diffquot(f,a,b,n): res=[] for i in [0..n-1]: stelle=a+(b-a)/n*i res.append(f.diff()(stelle)) return res /// }}} {{{id=25| f(x)=x^2 diffquot(f,0,4,4) /// [0, 2, 4, 6] }}}

Aufgabe 5

Schreibe eine Funktion, die eine an sie übergebene Liste (z.B. L = [78,9,1,3,5,2,5,12]) der Größe nach sortiert, ohne dabei die Funktion sorted() oder sort() zu verwenden.
Hinweis: Eine Möglichkeit besteht darin, dass alle nachfolgenden Elemente mit dem ersten, zweiten, ... verglichen und wenn erforderlich vertauscht werden.

{{{id=19| def sort(L): alles_richtig=false while(alles_richtig != true): alles_richtig = true; for i in [0..len(L)-2]: if L[i]>L[i+1]: alles_richtig=false temp=L[i] L[i]=L[i+1] L[i+1]=temp return L /// }}} {{{id=26| L = [78,9,1,3,5,2,5,12] sort(L) /// [1, 2, 3, 5, 5, 9, 12, 78] }}}

Aufgabe 6

Schreiben Sie einen einfachen Taschenrechner, der eine Addition berechnen kann.

Also eine Funktion, die als Eingabe einen String der Form '$\left< zahl \right > + \left< zahl \right >$' bekommt und die Summe dieser beiden Zahlen zurück gibt.
Hinweis: Benutzen Sie die Funktion $split()$ um an die Zahlen zu kommen, die dann mit $int()$ in Zahlen umgewandelt werden können

{{{id=22| def summator(ausdr): Operanden=ausdr.split('+') sum=0 for summand in Operanden: sum += int(summand) return sum /// }}} {{{id=27| summator('3+4+2') /// 9 }}}

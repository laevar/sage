Schreiben Sie einen einfachen Taschenrechner, der eine Addition berechnen kann.

Also eine Funktion, die als Eingabe einen String der Form '$\left< zahl \right > + \left< zahl \right >$' bekommt und die Summe dieser beiden Zahlen zurück gibt.

Hinweis:

Benutzen Sie die Funktion $split()$ um an die Zahlen zu kommen, die dann mit $int()$ in Zahlen umgewandelt werden können

{{{id=7| def add(string): operants=string.split('+') summe=0 for summand in operants: summe += int(summand) return summe /// }}} {{{id=20| add('4+5+3') /// 12 }}}

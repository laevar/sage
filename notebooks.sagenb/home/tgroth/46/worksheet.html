Aufgabe 5

Schreibe eine Funktion, die eine an sie übergebene Liste (z.B. L = [78,9,1,3,5,2,5,12]) der Größe nach sortiert, ohne dabei die Funktion sorted() oder sort() zu verwenden.
Hinweis: Eine Möglichkeit besteht darin, dass alle nachfolgenden Elemente mit dem ersten, zweiten, ... verglichen und wenn erforderlich vertauscht werden.

{{{id=19| #Mögl. Lösung (alternativ bubble sort..) def sortieren(L): n = len(L) for i in [0..n-1]: for j in [i+1..n-1]: if (L[j] < L[i]) : Lj= L[j] L[j] = L[i] L[i] = Lj return L sortieren([78,9,1,3,5,2,5,12]) /// [1, 2, 3, 5, 5, 9, 12, 78] }}} {{{id=21| #bubble sort def bubble(L): n = len(L) AnzahlVertauschungen = 1 while AnzahlVertauschungen > 0: AnzahlVertauschungen = 0 for i in [0..n-2]: if (L[i] > L[i+1]) : AnzahlVertauschungen += 1 dummy= L[i+1] L[i+1] = L[i] L[i] = dummy return L bubble([78,9,1,3,5,2,5,12]) /// [1, 2, 3, 5, 5, 9, 12, 78] }}}

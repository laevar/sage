\begin{aufg}[-1]
Entwickeln Sie mit den Interaktiven Möglichkeiten von Sage das Spiel \emph{Tic Tac Toe}. Dabei soll ein Spieler in der Lage sein
ein Kreuz zu setzen. Der vom Gegner gesetzte Kreis soll vom Computer (zufällig) gesetzt werden. Hat einer der beiden Spieler 
gewonnen, soll es eine entsprechende Meldung geben. 

Sie können auch versuchen den Computergegner mit einer rudimentären Intelligenz auszustatten. 

Anregungen, wie so ein Spielbrett und das Setzen von Kreuzen bzw. Kreisen funktionieren kann gibt es hier: 
\emph{http://wiki.sagemath.org/interact/games}
\end{aufg}

\begin{aufg}
Schreiben Sie eine Prozedur, deren Eingabeparameter zwei Strings sind
und die den kleinsten Index berechnet, ab dem der Erste dieser Strings
im Zweiten enthalten ist.  Kann der String nicht gefunden werden, so
soll die Prozedur \verb+False+ zurückgeben.
\end{aufg}

\begin{aufg}
Schreiben Sie eine Prozedur, deren Eingabeparameter zwei Strings und
ein Index sind und die den ersten String am gegebenen Index in den
Zweiten einfügt.
\end{aufg}

\begin{aufg}
Schreiben Sie eine Prozedur, die für einen String -- z.B. einen
Sage-Ausdruck in String-Form -- überprüft, ob alle geöffneten Klammern
auch wieder geschlossen werden.
\end{aufg}

\begin{aufg}
Betrachten Sie das Newton-Verfahren zum Berechnen der Nullstellen von
$z^3-1$, d.h.
\begin{equation*}
  z_{n+1}=z_n-\frac{z_n^3-1}{3z_n^2}
\end{equation*}
mit einem Startwert $z_0=x+iy$. Schreiben Sie eine Funktion, die zu
gegebenem $x,y\in\mathbb{R}$ überprüft, gegen welche der drei Lösungen
von $z^3-1=0$ das Newton-Verfahren konvergiert.  Sie können von
Konvergenz ausgehen, wenn $|z_n^3-1|<0.1$ ist und in dem Fall das
Argument von $z_n$ als Ergebnis zurückgeben.  Sollte nach 20
Schritten keine Konvergenz eintreten, geben Sie 0 zurück.

Erstellen Sie dann einen Plot der Funktion analog zur Mandelbrot-Menge
in der Vorlesung für $-1\leq x,y \leq 1$ mit \verb+plot_points=60+.
\end{aufg}

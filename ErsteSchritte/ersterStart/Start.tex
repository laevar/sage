Egal ob Sage nun an den Rechnern in der Mathe genutzt wird oder eine eigene Installation verwendet wird. Die ersten Schritte sind die gleichen.
Gestartet wird Sage entweder über den entsprechenden Menüeintrag oder über die Konsole. 
In der Konsole muss zum Starten der Befehl \verb sage  eingegeben werden. Nun wird ein Passwort festgelegt. Ist das Programm gestartet,
erkennbar daran, dass in der Konsole \emph{sage:} steht kann die grafische Oberfläche durch Eingeben des Befehls \verb notebook()  gestartet 
werden. Danach sollte sich automatisch ein Internetbrowser mit der Sagenotebook Oberfläche öffnen. Sollte dies einmal nicht passieren, kann dies 
auch manuell erledigt werden. Also einen Browser starten und die angegebene Adresse eingeben. 
\paragraph{}
Ist die Oberfläche zu sehen, kann begonnen werden, indem ein neues Arbeitsblatt durch Klick auf \emph{new Worksheet} geöffnet wird. Es kann
in dem folgenden Dialog dann dem Dokument noch ein anderen Namen gegeben werden. 
Nun kann es aber wirklich losgehen. In den Eingebefenstern können die Sagebefehle direkt eingegeben werden und ein Klick auf \emph{evaluate} 
führt diesen Code dann aus. Alternativ geht dies auch mit der Tastenkombination \emph{Shift(Großschreib)+Enter}. Zum Beispiel können die Berechnungen
\verb 1+1  oder \verb sin(2)  ausgewertet werden. 
Die Änderungen an dem Dokument können dann oben rechts gespeichert oder auch verworfen werden. Wählt man zum Beispiel den Eintrag 
\emph{Save \& quit} wird gespeichert und man landet wieder auf der Startseite bei der unter \emph{aktive Dokuments} nun ein neuer Eintrag zu sehen
sein sollte. Ein Klick darauf ermöglicht das Weiterarbeiten an dieser Datei. 

Weiteres zur Bedienung und möglichen Befehlen gibt es dann in der Vorlesung.

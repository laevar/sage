\begin{aufg}
Schreiben Sie eine Prozedur, die eine Nullstelle einer  gegebenen Funktion $f:[a_0,b_0]
\rightarrow 
\mathbb{R}$ mit $f(a_0)\cdot f(b_0)<0$  zu einer gegebenen Genauigkeit $TOL$ berechnet. Hierzu betrachten wir eine Folge von
Intervallen $[a_k,b_k]$, die sich aus der folgenden Vorschrift ergeben: \\

Wiederhole bis $|b_0-a_0|\leq 2^{k+1}*TOL$
\begin{itemize}
\item Berechne $g:=f(\frac{a_k+b_k}{2})$.
\item Gilt $f(a_k)*g<0$ so definiere $a_{k+1}:=a_k$,
  $b_{k+1}:=\frac{a_k+b_k}{2}$.
\item Andernfalls berechne  $a_{k+1}:=\frac{a_k+b_k}{2}$,
  $b_{k+1}:=b_k$. 
\end{itemize}  
Testen Sie die Prozedur an $f(x):= x^3 - 2$, $[a_0,b_0]:=[0,3]$ und für
$TOL=10^{-4}$ und $TOL=10^{-6}$. Verwenden Sie maximal $1000$ Iterationen.
\end{aufg}
